# 刚刚好的学习笔记

- 刚刚好，一个基于微服务架构的前后端分离博客系统。前台使用 Vue + Element , 后端使用 spring boot + spring cloud + mybatis-plus 进行开发，使用 Jwt + Spring Security 做登录验证和权限校验，使用 ElasticSearch 和 Solr 作为全文检索服务，使用 Github Actions 完成博客的持续集成，文件支持上传七牛云。
